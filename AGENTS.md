# Agents

This project's learning memory is an in-repo git mind on `refs/mind/main` — read `ONEMIND.md` and
follow its protocol. Index ref `refs/mind/HEAD` tracks the latest thought. ONEMIND.md LIMIT 1GB

## How to use the mind

The mind is already initialized. You don't need permission or a prompt — use it naturally:

1. **At session start:** run the four recap commands in ONEMIND.md, then open with what's relevant.
2. **Before acting:** search the mind for prior context on what the user is asking about.
3. **During work:** when the user answers a question, states a preference, or you discover
   something non-obvious — record it immediately. See "What's worth recording" in ONEMIND.md.
4. **After completing a task:** if the outcome was surprising or the approach changed, log it.

All mind commands use git plumbing only — never checkout, never working-directory files.
